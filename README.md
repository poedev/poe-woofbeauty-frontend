# poe-template-frontend-gulp
A FRONTEND TEMPLATE WITH GULP (TASK RUNNER)

# LANGUAGE
- JADE
- LESS
- COFFEE-SCRIPT
- HTML, CSS, JS

# LIBRARY
- Bootstrap
- jQuery

# HOW TO USE
- Config paths and processes in gulpfile.js

## STEP 1: INSTALL MODULES
run command
```
npm install
```

## STEP 2: INSTALL GULP
run command
```
npm install -g gulp
```
or
```
npm install gulp
```

## STEP 3: RUN GULP
BUILD FRONTEND FILES
```
gulp
```
RUN A BASIC SERVER
```
gulp server
```
